Palomar Knot

  1. Thread the line through the eye of the hook or lure
  2. Run the tag end back through the eye in the opposite direction
  3. Form a loop with the tag end
  4. Use the loop to tie an overhand knot with the doubled line
  5. Pass the hook or lure through the loop
  6. Moisten the line
  7. Pull the tag end and standing line to tighten the knot
  8. Trim the tag end

Uni Knot

  1. Thread the line through the eye of the hook or lure
  2. Lay the tag end over the main line
  3. Form a loop with the tag end
  4. Wrap the tag end around the doubled line 6 to 7 times
  5. Moisten the line
  6. Pull on the tag end until it's tight
  7. Pull on the main line until the knot locks into place
  8. Trim the tag end

San Diego Jam Knot

  1. Thread the line through the eye of the hook or lure
  2. Create a loop
  3. Open the loop
  4. Wrap the tag end
  5. Pass the tag end through the base of the knot
  6. Thread the tag end through the top of the knot
  7. Moisten the line
  8. Tighten the knot
